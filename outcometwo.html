2.1 Review the ongoing use of IT tools and change approach as needed – in making your own website, are you happy with the tools you used? E.g. Visual Studio Code, Trello.

Visual Studio Code is the main syntax programme that I am using. Its software enables me to write syntax in HTML, CSS and JavaScript in order to achieve the end result - my own website. Visual Studio Code features a source code editor, which is perfect for me to use. VS Code helped me be extremely productive with syntax highlighting, bracket-matching, auto-indentation, and more.

I've tested and retested the code I've written within the Chrome browser. It is extremely important to view your website in a test browser whilst you’re coding so that you can see the live changes you make otherwise it would be very difficult to judge what the end product will look like.

Trello is a collaboration tool that organises your projects into boards. In one glance, Trello tells you what's being worked on, who's working on what, and where something is in a process. Imagine a cork board, filled with lists of post-it notes, with each note as a task for you and your team.

The tools I selected were definitely appropriate as VS code is a great software tool for writing HTML and CSS. Following my attendance at the lectures on VS code, I felt quite at home.
